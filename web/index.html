📚 API Reference

Core Functions:
cu.init() - Initialize Lua VM
cu.compute(code) - Execute Lua code
cu.getBufferPtr() - Get buffer address
cu.getBufferSize() - Get buffer size (64KB)
cu.getMemoryStats() - Get memory info
cu.runGc() - Run garbage collection

_io Table API:
cu.setInput(data) - Set _io.input with JS object
cu.getOutput() - Get _io.output as JS object
cu.clearIo() - Clear all _io contents

Persistence:
cu.saveState() - Save _home to IndexedDB
cu.loadState() - Load _home from IndexedDB

(Also available as lua.* for backward compatibility)
